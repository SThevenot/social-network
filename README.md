# Social Network Back-End Application 
  ![](https://img.shields.io/badge/license--green)

  #### The goal of this projext was to create a basic API for a social networking application where users can share thoughts, reactions, and also have friends. Using Express.js and MongoDB this application will render all the users in the databse, create, update, and delete users and add friends. The same functions apply to thoughts as well.
